---------------------------------------------------------------------------
Index creates an entry for each value of the indexed columns
---------------------------------------------------------------------------
By default, index type is BTREE.
---------------------------------------------------------------------------
Types of an index based on the storage engine of the table.
---------------------------------------------------------------------------
SN	Storage Engine			Index Type
1.	InnoDB				BTREE
2.	Memory/Heap			HASH, BTREE
3.	MYISAM				BTREE
