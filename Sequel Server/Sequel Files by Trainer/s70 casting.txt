-----------------------------------------------------------------------------
			SQL Server CAST
-----------------------------------------------------------------------------
Data type conversion is a method used to change the data type of a value into the other type. 
-----------------------------------------------------------------------------
SQL Server can convert the data type of a value in two ways: 
-----------------------------------------------------------------------------
		a.	implicitly 
-----------------------------------------------------------------------------
		b.	explicitly. 
-----------------------------------------------------------------------------
SELECT 10 + '10' AS Result  
go
-----------------------------------------------------------------------------
Result
-----------------------------------------------------------------------------
20
-----------------------------------------------------------------------------
			1. CAST Function to convert Decimal to Integer
-----------------------------------------------------------------------------
SELECT CAST(10.95 AS INT) AS Result; 
go
-----------------------------------------------------------------------------
Result
-----------------------------------------------------------------------------
10
-----------------------------------------------------------------------------
			2. CAST Function to convert Decimal into another 				Decimal with different length
-----------------------------------------------------------------------------
SELECT CAST(10.95 AS DEC(3, 0)) AS Result; 
go
-----------------------------------------------------------------------------
Result
-----------------------------------------------------------------------------
11
-----------------------------------------------------------------------------
When we convert a data type value in different positions, SQL Server gives a truncated or rounded result according to the following rules:
-----------------------------------------------------------------------------
			3. CAST Function to convert String to DATETIME
-----------------------------------------------------------------------------
SELECT CAST('2021-04-26' AS DATETIME) AS Result;  
go
-----------------------------------------------------------------------------
 Result
-----------------------------------------------------------------------------
2021-04-26 00:00:00.000
-----------------------------------------------------------------------------
			4. CAST Function to convert DATETIME to Varchar
-----------------------------------------------------------------------------
SELECT CAST(GETDATE() AS VARCHAR) AS Result;
go
-----------------------------------------------------------------------------
Result
-----------------------------------------------------------------------------
Oct  8 2024  3:48PM
-----------------------------------------------------------------------------
			5. CAST Function with a NULL value
-----------------------------------------------------------------------------
DECLARE 	@strValue AS VARCHAR(50)  
SET 		@strValue = NULL  
SELECT 		CAST(@strValue AS INT) AS Result1;  
SELECT 		CAST(NULL AS INT) AS Result2;  
go
-----------------------------------------------------------------------------
Result1
-----------------------------------------------------------------------------
NULL
-----------------------------------------------------------------------------
Result2
-----------------------------------------------------------------------------
NULL
-----------------------------------------------------------------------------
			TRY_CAST Function
-----------------------------------------------------------------------------
It is the advanced form of the CAST function , handles errors, while casting. 
-----------------------------------------------------------------------------
SELECT CAST('mba' AS INT) AS Result; 
go 
-----------------------------------------------------------------------------
Msg 245, Level 16, State 1, Line 16
Conversion failed when converting the varchar value 'mba' to data type int.
-----------------------------------------------------------------------------
SELECT TRY_CAST('mba' AS INT) AS Result;  
go
-----------------------------------------------------------------------------
Result
-----------------------------------------------------------------------------
NULL
-----------------------------------------------------------------------------
